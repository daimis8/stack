# Stack
This project shows how stack works in C.

## Run Locally

Clone the project:

```bash
  git clone git@github.com:daimis8/JavaGradesProject.git
```

Go to the project directory:

```bash
  cd JavaGradesProject
```

Run the makefile:

```bash
  make
  make run
```

Cleaning .o and .exe files:

```bash
  make clean
```
